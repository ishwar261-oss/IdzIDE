# IdzIDE

A lightweight and user-friendly code editor built with C# and Windows Forms.

## Features
- Create, Open, and Save files
- Dark Theme UI
- Line and Character Counter
- Automatic Bracket Completion
- Clean and Simple Interface

## Technologies Used
- C#
- .NET
- Windows Forms

## Getting Started
1. Clone the repository.
2. Open the solution in Visual Studio.
3. Build and run the project.
